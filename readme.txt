Обучение
Для запуска обучения:
1. Проверьте что в корне проекта есть файл config_model.yaml и config_yolo.yaml, если их нет перенесите из директории default_config.
2. Датасет в формате YOLO разместите в директории dataset.
  2.1 Пример расположения папок dataset:
        - dataset
            - test
                - images
                    - 1.jpg
                    ...
                - labels
                    - 1.txt
                    ...
            - train
                - images
                - labels
            - valid
                - images
                - labels
            - data.yaml
3. Отредактируйте файл data.yaml указав абсолютные пути до директорий
    Пример:
    train: /home/user/UAV_AI/UAV_AI_yolo/dataset/train/images
    val: /home/user/UAV_AI/UAV_AI_yolo/dataset/valid/images
    test: /home/user/UAV_AI/UAV_AI_yolo/dataset/test/images
4. Отредактируйте файл config_model.yaml и config_yolo.yaml, который находится в родительской директории
 указав необходимые параметры которые будет применены к модели в процесси обучения.
5. Для запуска процесса обучения нажмите ПКМ по файлу start_train.sh
в родительской директории и выберите в выпадающем меню пункт "Запустить в Konsole".
6. Дождитесь окончания процесса обучения.

--------------------------------
Тестирование
1. Для тестирования перенесите обученную модель которая
   находится в директории runs/detect/<название модели>/weigths/bets.pt в родительский каталог
2. Загрузите видео для тестирования в папку dataset/test, видео должно иметь название video.mp4
3. Для запуска процесса детекции нажмите ПКМ по файлу start_detect.sh
в родительской директории и выберите в выпадающем меню пункт "Запустить в Konsole".
4. Визуально оцените результат обучения.
